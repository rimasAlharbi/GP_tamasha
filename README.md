# Tamsha - Smart Location Picker

Tamsha is a web-based Smart Location Picker designed to help users discover nearby places, plan outings, and generate optimized itineraries.

## Features
- Interactive map using Leaflet.js
- Smart place filtering
- AI-powered recommendations
- Route optimization
- User authentication
- Reviews and ratings
- Admin dashboard

## Technologies
- HTML
- CSS
- JavaScript
- Supabase
- Leaflet.js
- OpenStreetMap
- Groq AI (Llama 3.3)

## Project Description
Tamsha simplifies outing planning by allowing users to explore nearby cafés, restaurants, and entertainment venues, apply smart filters, receive AI recommendations, and generate optimized routes through an interactive web application.
